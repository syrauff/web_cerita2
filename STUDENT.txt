TEKNOLOGI YANG DIGUNAKAN
═══════════════════════════════════════════════════════════════════════

- JavaScript ES6+ (Classes, Private Fields, Arrow Functions, Async/Await)
- HTML5 Semantic Elements
- CSS3 (Grid, Flexbox, Animations, Media Queries, CSS Custom Properties)

- IndexedDB (native API, tanpa library tambahan)
- View Transitions API
- MediaDevices API (WebRTC camera)
- LocalStorage (auth token)
